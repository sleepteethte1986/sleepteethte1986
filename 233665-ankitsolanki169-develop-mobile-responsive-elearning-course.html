Develop Mobile Responsive Elearning Course











About:

I am specialist in creating mobile responsive elearning course using adobe captivate 2019 tool.



What I would need from you?

I would need storyboard/script of the course in word document or powerpoint which should contain:-

1) Title name of each slide

2) Contain text for each slide

3) Image assets (if you have otherwise I will take care of it)

4) Interactivity instruction (not mandatory if you don't have I can take care of it as per the content provided)

5) Voiceover files (if you don't have I can use adobe captivate voiceover only)



I don't create elearning courses from ebook.



Please discuss about your requirements through inbox before creating the order.
